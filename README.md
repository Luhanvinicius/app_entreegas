# App Entregas Modernizado 🚀

Um ecossistema completo de delivery (API, Web e Mobile) modernizado com foco em experiência do usuário e agilidade logística.

## ✨ Principais Melhorias:

### 📱 Mobile (Entregador)
- **Alertas Estilo Uber/iFood**: Pop-up imersivo com som de notificação para novos pedidos, garantindo que o entregador nunca perca uma corrida.
- **Sessão Persistente**: Login que não expira ao fechar o app (uso de AsyncStorage).
- **Logística Refinada**: Exibição clara de endereços de **Retirada** (na loja) e **Entrega** (no cliente) com ícones distintos.
- **Navegação Moderna**: Troca de gaveta lateral (drawer) por abas inferiores (Bottom Tabs) para maior estabilidade e uso intuitivo.

### 🌐 Web (Admin e Lojista)
- **Painel Administrativo Completo**: Gestão full (Editar/Excluir) de Lojas e Entregadores.
- **Perfil do Lojista**: Interface para o dono da loja configurar seu endereço e dados bancários/contato.
- **Design Premium**: Interface revitalizada com Lucide Icons e tipografia moderna.

### ⚙️ Backend (API)
- **Prisma Schema Atualizado**: Suporte a endereços de loja.
- **Novos Endpoints**: CRUD completo de usuários, lojas e monitoramento de saldo.
- **Segurança**: Autenticação via JWT com middleware de roles.

## 🛠️ Tecnologias:
- **Frontend**: React, Lucide Icons, Axios.
- **Mobile**: Expo (SDK 54+), React Native, expo-av, lucide-react-native.
- **Backend**: Node.js, Express, Prisma (SQLite), TypeScript.
